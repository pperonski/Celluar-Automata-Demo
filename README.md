# Celluar automata demo

A simple program simulating celluar automata written in python using torch.

You can specify your own kernel with activation function, enjoy.

Before executing a program, create virtual environment:

``` bash
python3 -m venv .venv
source .venv/bin/activate
```

install dependencies:

``` bash
pip install -r ./requirments.txt
```

then execute program by typing:

``` bash
python3 main.py
```
